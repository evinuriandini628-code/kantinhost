# Assets KantinHost

Folder ini dipakai untuk menyimpan semua asset visual KantinHost.

## Rekomendasi Asset

```txt
public/assets/
├── kantinhost-logo.png       # Logo utama KantinHost
├── kantinhost-icon.png       # Icon / favicon KH
├── kantinhost-horizontal.png # Logo horizontal untuk navbar/footer
├── kh-fox-default.png        # Maskot utama
├── kh-fox-avatar.png         # Avatar default user
├── kh-fox-working.png        # Maskot working/loading
├── kh-fox-support.png        # Maskot support
├── kh-fox-sad.png            # Maskot empty/error state
└── server-iso.png            # Icon kotak server
```

## Catatan

Gunakan format PNG transparan kalau memungkinkan agar mudah dipakai di dark UI dan card glassmorphism.
