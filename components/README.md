# Components

Folder ini untuk komponen reusable KantinHost.

## Rekomendasi Isi

```txt
components/
├── Navbar.jsx          # Navbar desktop/mobile
├── Sidebar.jsx         # Sidebar dashboard
├── GlassCard.jsx       # Card glassmorphism reusable
├── ServerCard.jsx      # Card server user
├── StatCard.jsx        # Card statistik dashboard
├── BottomNav.jsx       # Navigasi mobile
└── Layout.jsx          # Layout utama dashboard/client area
```

## Style

- Dark UI
- Neon blue accent
- Glassmorphism
- Rounded corners
- Responsive mobile-first
