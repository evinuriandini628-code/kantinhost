# Pages

Folder ini untuk halaman Next.js KantinHost.

## Rekomendasi Halaman

```txt
pages/
├── index.jsx           # Landing / home
├── login.jsx           # Login user
├── register.jsx        # Register + OTP WA
├── dashboard.jsx       # Ringkasan client area
├── servers.jsx         # List server user
├── server/[id].jsx     # Detail server Pterodactyl
├── claim.jsx           # Claim/link server via token
├── billing.jsx         # Billing, invoice, renewal
├── tickets.jsx         # Support ticket
└── account.jsx         # Profile & setting akun
```

## Fokus Produk

KantinHost lebih fokus ke client area dan Pterodactyl server manager, bukan sekadar katalog hosting umum.
